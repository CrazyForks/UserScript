## 1. 如何安装/使用脚本？

要使用任何脚本，首先需要浏览器安装 **Tampermonkey** 脚本管理器扩展，比如 [Chrome](https://chromewebstore.google.com/detail/jinjaccalgkegednnccohejagnlnfdag)。
